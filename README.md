# AQI_Analysis_Dashboard
End-to-end Power BI AQI dashboard analyzing air quality trends, pollutant impact, and KPIs with interactive filters for state, year, and area, showcasing data cleaning, DAX, and professional data visualization skills.

Developed an end-to-end AQI Dashboard using Power BI to analyze air quality trends and identify key pollutants impacting environmental health. The project focuses on transforming raw AQI data into meaningful insights through data modeling, DAX, and interactive visualizations.

**Key Features & Work Done**

1.Imported and cleaned AQI datasets to ensure accuracy and consistency

2. Categorized AQI levels into Good, Moderate, Unhealthy, and Hazardous

3. Created DAX measures to calculate average AQI, KPIs, and trend metrics

4. Analyzed prominent pollutants contributing to poor air quality

5. Built interactive dashboards with slicers for State, Year, and Area

6. Designed professional, insight-driven visuals for quick decision-making

**Tools & Skills Used**
1. Power BI

2. DAX

3. Data Cleaning & Transformation

4. Data Modeling

5. Dashboard Design & Data Visualization

**Project Outcomes**

1. Enabled clear comparison of air quality across regions and time periods

2. Helped identify pollution hotspots and major contributing pollutants

3. Delivered a user-friendly dashboard for data-driven environmental insights
